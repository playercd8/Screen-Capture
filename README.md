# Screen-Capture

2003年拿書上的翻譯改寫的抓圖工具

##授權

MIT License

##使用說明

1.使用按鍵【ScrollLock】切換視窗模式或全螢幕模式。

2. 預設使用按鍵【F9】抓圖 (可以改按鍵設定)
